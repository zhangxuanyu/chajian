脚本执行方法：
bt_Util.executeScript(scp);
scp为脚本，返回值为一数组对象，在本文档中，以Log0表示返回的数组对象的首个元素，Log1为第二个元素，依此类推。

---------------------------------------------------------------------------

一个典型的脚本执行命令如下：
bt_Util.executeScript("Render\\Camera\\WorldToScreen 1.0 2.0 3.0;");
该脚本执行后将返回世界坐标系中点(1.0, 2.0, 3.0)在屏幕坐标系中的位置

---------------------------------------------------------------------------

本文档中，但凡以方括号对"[]"括起来的部分，均为参数变量，比如上述脚本的说明文档如下：
Render\Camera\WorldToScreen [Wx] [Wy] [Wz];

作用：
计算世界坐标系中的点在屏幕坐标系中的位置

参数：
[Wx] - double，点在世界坐标系中的X坐标
[Wy] - double，点在世界坐标系中的Y坐标
[Wz] - double，点在世界坐标系中的Z坐标

返回值：
Log0 : [Sx] [Sy] [Sz]
[Sx] - int，点在屏幕坐标系中的X坐标（以像素为单位）
[Sy] - int，点在屏幕坐标系中的Y坐标（以像素为单位）
[Sz] - float，点在屏幕坐标系中的深度

---------------------------------------------------------------------------

每条脚本均以分号（";"）结尾，bt_Util.executeScript可一次性执行多条脚本（每条脚本间以";"分隔），但是由于返回值只有一个对象，其多个返回值难以区隔，因此不建议一次执行多条脚本。
