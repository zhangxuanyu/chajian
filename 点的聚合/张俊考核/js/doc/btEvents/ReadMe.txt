事件监听：
bt_PlugManager.addEventListener(e, f);
e为欲监听的事件
f为事件处理函数

---------------------------------------------------------------------------

解除事件监听：
bt_PlugManager.removeEventListener(e, f);

---------------------------------------------------------------------------

事件处理函数形如：
var f = function(ep) {
}
其中ep为事件参数，是一个数组对象，本文档中将以ep0表示该对象首个元素，ep1表示第二个元素，依此类推
